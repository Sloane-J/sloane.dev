# Samuel Dorkey — Portfolio

Personal portfolio website for Samuel Dorkey, a software developer based in Ghana. Built with Astro and React, deployed on Vercel.

**Live:** [samuel-dorkey.vercel.app](https://samuel-dorkey.vercel.app)

---

## About

This is a production portfolio site showcasing selected projects, services, and client work. It is designed with a brutalist aesthetic — sharp edges, monospace type, and a dark colour palette — and built for performance with minimal JavaScript.

---

## Tech Stack

| Layer | Tool |
|---|---|
| Framework | [Astro](https://astro.build) |
| UI Components | [React](https://react.dev) |
| Styling | [Tailwind CSS](https://tailwindcss.com) |
| Animation | [Framer Motion](https://www.framer.com/motion) |
| Smooth Scroll | [Lenis](https://lenis.darkroom.engineering) |
| Fonts | Google Fonts — Playfair Display, Inter |
| Analytics | [Umami](https://umami.is) |
| Deployment | [Vercel](https://vercel.com) |

---

## Features

- Brutalist dark design system with `#FF5733` orange accent
- Scroll-snapped project showcase with image sliders
- Smooth page scroll via Lenis
- View transitions between pages
- Sitemap auto-generation via `@astrojs/sitemap`
- Mobile-first responsive layout
- SEO meta tags, Open Graph, canonical URLs
- Google Search Console and Bing Webmaster verified
- Reduced motion support

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) installed
- Node.js 18+

### Install

```bash
git clone https://github.com/Sloane-J/sloane.dev.git
cd sloane.dev
bun install
```

### Dev server

```bash
bun run dev
```

Open [http://localhost:4321](http://localhost:4321)

### Build

```bash
bun run build
```

### Preview build

```bash
bun run preview
```

---

## Deployment

To deploy manually:

```bash
bun run build
# then push to main
```
